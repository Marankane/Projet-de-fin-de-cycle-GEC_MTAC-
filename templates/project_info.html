Plateforme de Gestion Électronique des Courriers - MINTRANSPORT Niger

Architecture Générale

Une plateforme Django complète pour gérer le cycle de vie des courriers administratifs :

  • Interface web sécurisée et responsive
  • Base de données PostgreSQL robuste
  • API REST avec JWT
  • Système de rôles et permissions
  • Journal d'audit immuable
5 Rôles Utilisateurs
Rôle Fonction
Agent Courrier Reçoit, scanne, enregistre
Secrétaire Consultation, répartition
Chef Service Instructions, validation
Directeur Vision globale, stats
Administrateur Gestion complète
Workflow Métier (3 Validations)

Un courrier traverse 3 validations avant clôture :

1

Agent Courrier → SGA

Enregistrement + scan du courrier physique

2

SGA → SG

Avis du Secrétaire Général Adjoint (positif/négatif)

3

SG → Ministre

Avis du Secrétaire Général (positif/négatif)

4

Ministre → Clôture

Décision finale et clôture du courrier
